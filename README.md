# Greek Social Media Language Analysis

## Overview
This project analyzes Greek social media language using basic NLP techniques.

## Objectives
- Explore informal Greek online communication
- Identify word frequency patterns
- Analyze emotional expression through emojis

## Findings
- Frequent use of emotional and informal language
- Laughter expressions like "χαχα" are common
- Emojis act as emotional/pragmatic markers

## Tools
- Python
- Google Colab
- Collections (Counter)

- greek_analysis.ipynb → main analysis notebook
